About NewsVisualizer

What is NewsVisualizer?

NewsVisualizer is a news source comparison tool. Its purpose is to provide an objective, multi-dimensional viewpoint of the sources present, allowing you, as the user, to make your own judgement on the reliability of each source. NewsVisualizer makes use of a large dataset of articles, which was collected each day over a six month-long continuous time span. The main idea behind the tool's visualization is that each news source can be represented by the collection of its published articles. NewsVisualizer aims to provide a true representation of each source, doing so by displaying the minimum, maximum, average and full range of values for any of the given article features. NewsVisualizer uses a large set of article features with the intention of providing a complete viewpoint to analyze the reliability of a source.

What are the features used?

NewsVisualizer uses a set of article features, which are generally based on the word counts within the articles. Specifically, NewsVisualizer makes use of many of the features provided by Linguistic Inquiry and Word Count (LIWC). For example, the feature 'Perception Words' is an average number of discrepancy words that the article uses; these words include "should", "would, "could", etc. The full list of features and a detailed description of each can be found here .

How can I add more sources?

The NewsVisualizer github provides all the code used in creating this tool. Given your own crawler or article collection, you can compute the features for each article and add it to the tool's database.
